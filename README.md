# Central Cilada 🚴

Site oficial do **Pedal Cilada**, grupo de ciclismo da Vila de Carapajó, em Cametá-PA.

## Sobre

Site estático (HTML, CSS e JavaScript) com:

- **Início + Sobre** — apresentação do grupo
- **Calendário de pedais** — próximos passeios
- **Galeria de fotos** — momentos na estrada

## Como editar

Tudo que muda com frequência fica no topo do arquivo [`script.js`](script.js):

- **Pedais** → edite a lista `PEDAIS` (data, hora, local, distância, nível e descrição)
- **Fotos** → edite a lista `FOTOS` (troque os `src` pelas fotos reais em `img/`)

Textos, cores e links ficam em [`index.html`](index.html) e [`styles.css`](styles.css).

## Publicação

Hospedado na [Vercel](https://vercel.com). Cada alteração enviada ao GitHub publica automaticamente.
